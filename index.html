ALGORITHM

Menggabungkan akurasi algoritma YIN dengan stabilitas Kalman Filter. Hasilnya? Jarum yang tidak "gemetar" dan deteksi pitch yang instan, bahkan untuk senar Low-B bass.

ARCHITECTURE

Memory Efficient

Dibuat seringan mungkin. Tanpa bloatware, tanpa library iklan yang memakan RAM. Fokus CPU 100% didedikasikan untuk analisis audio.

The Privacy Manifesto

Aplikasi tuner tidak seharusnya memata-matai penggunanya. Di Query Cat, kami menganut prinsip "Hostile Privacy".

Ephemeral Processing

Audio dari mikrofon hanya hidup di RAM selama 23 milidetik untuk dihitung frekuensinya, lalu langsung dihapus/overwrite. Tidak pernah disimpan ke storage.

Air-Gapped Logic

Kode inti C++ tuner tidak memiliki modul networking. Secara teknis, engine audio kami mustahil mengirim data ke internet.
