The Privacy Manifesto

Kami menganut prinsip "Hostile Privacy".

Aplikasi utilitas tidak seharusnya memata-matai penggunanya. Berikut adalah janji teknis kami:

Ephemeral Processing

Audio dari mikrofon hanya hidup di RAM selama ~23ms untuk kalkulasi, lalu langsung di-overwrite. Tidak pernah disimpan ke storage.

Air-Gapped Logic

Kode inti C++ tuner tidak memiliki modul networking. Secara teknis, engine audio kami mustahil mengirim data ke internet.

Frequent Queries

Jawaban teknis untuk musisi kritis.

Mengapa "Native C++" lebih baik dari tuner biasa?

Tuner berbasis Java/Kotlin bergantung pada "Garbage Collector" yang menyebabkan jeda acak (stutter). C++ memberikan kontrol memori manual, menjamin aliran audio yang mulus dan latensi ultra-rendah.

Apakah support Drop Tunings?

Tentu. PurrTune mendukung Standard, Drop D, Half-Step Down (Eb), Open G, DADGAD, dan Full Chromatic Mode untuk tuning custom apapun.

Apakah perlu koneksi internet?

Sama sekali tidak. Seluruh pemrosesan Digital Signal Processing (DSP) dilakukan lokal di chipset HP Anda (On-Device). Mode pesawat aman.

HP saya spek kentang, kuat gak?

PurrTune dioptimasi untuk berjalan mulus bahkan di perangkat entry-level (Android 8.0 ke atas) dengan konsumsi RAM di bawah 50MB.
